# Staff Management

Simple staff management application built with ASP.NET Core Web API and React.

## Features

- Add, edit, delete, and list staff records.
- Staff fields:
  - Staff ID: string, 8 characters, generated by the backend as `STF00001`.
  - Full name: string, max 100 characters.
  - Birthday: date.
  - Gender: integer, `1 = Male`, `2 = Female`.
- Advanced search by Staff ID, Gender, and Birthday from/to.
- Export advanced search results to Excel and PDF.
- User-friendly validation messages and basic security headers.

## Tech Stack

- Backend: ASP.NET Core Web API, C#, EF Core, SQLite.
- Frontend: React + Vite.
- Backend tests: NUnit unit tests and integration tests.
- End-to-end tests: Node test runner against the live backend and frontend proxy.
- CI: GitHub Actions.

## Run Locally

Start the backend:

```bash
cd Backend/StaffManagement
dotnet run --urls http://localhost:5137
```

Start the frontend:

```bash
cd Frontend
npm install
npm run dev -- --host 127.0.0.1
```

Open:

```text
http://127.0.0.1:5173
```

## Test

Backend:

```bash
cd Backend
dotnet test Backend.sln
```

Frontend:

```bash
cd Frontend
npm run lint
npm run build
```

End-to-end:

```bash
cd Frontend
npm run test:e2e
```

## API Summary

- `GET /api/Staffs`
- `GET /api/Staffs/{staffId}`
- `POST /api/Staffs`
- `PUT /api/Staffs/{staffId}`
- `DELETE /api/Staffs/{staffId}`
- `GET /api/Staffs/search`
- `GET /api/Staffs/export/excel`
- `GET /api/Staffs/export/pdf`
